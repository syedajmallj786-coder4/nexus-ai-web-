# NEXUS REALM

A full-stack entertainment platform dedicated to Anime, Movies, and Games, built with a dark cyberpunk and Japanese neon aesthetic.

## Features
- Browsing categories (Anime, Movies, Games)
- "Now Trending" and "New" badges
- Watchlist saving (localStorage)
- Real-time search/filtering
- Responsive & beautifully animated UI using glassmorphism

## Tech Stack
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Backend**: Node.js, Express.js
- **Design Concept**: Cyberpunk / Neon / Glassmorphism

## How to Run Locally

### 1. Start the Backend API
```sh
cd backend
npm install
npm start
```
The server will start on `http://localhost:3000`.

### 2. View the Frontend
Simply open `frontend/index.html` in your web browser. Or, you can serve it via a local development server like VS Code Live Server for the best experience.
